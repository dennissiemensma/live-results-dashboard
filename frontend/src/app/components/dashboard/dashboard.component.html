Mass Start Settings
s
Max seconds gap between groups
Minimum time difference (in seconds) between two competitors on the same lap for them to be placed in separate standings groups. Lower values create more groups; 0 puts every competitor in their own group.
Max groups
Maximum number of standings groups to display in the group strip. Competitors beyond the last group are collected into a "Tail of the race" group. Set to 0 to disable grouping entirely.
%
Lap variance threshold
Percentage difference between consecutive lap times used to colour lap badges. Within threshold → green. Slower by more than threshold → orange. Faster by more than threshold → purple.
E.g. at 5%: a previous lap of 30.0 s means slower > 31.5 s turns orange, faster < 28.5 s turns purple.
Show lap times
Show or hide the lap time badge strip on each mass-start competitor row. Hiding lap times gives a cleaner view during busy races.
